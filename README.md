## Project Description

BMI Calculator Pro is a Python desktop application developed using Tkinter. It allows users to enter their name, weight, and height, then calculates their Body Mass Index (BMI) and displays the corresponding health category.

The application also stores BMI records in a CSV file and provides graphical visualization of BMI category history using Matplotlib.

---

## Features

- Calculate Body Mass Index (BMI)
- Display BMI category
- Color-coded result display
- Save BMI history to CSV
- View BMI history
- Bar chart visualization
- Pie chart visualization
- User-friendly Tkinter GUI
